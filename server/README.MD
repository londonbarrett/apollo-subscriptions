# APOLLO SUBSCRIPTIONS

This is an exercise for implementing GraphQL subscriptions manually with
Apollo Server 2, Postgres and NodeJS. It is a basic message stream with
channels.

## Features

- Apollo Server 2.
- Docker.
- Postgres.

## Requirements

- Docker.

## Getting Started

- Clone the repository.
- Run docker compose.
